🖼️ Background Remover App - Streamlit Based
------------------------------------------------------

🧑‍💻 Developed By: Bhupendra Singh
📅 Year: 2025
🌐 Description:
This is a simple and powerful web app built using Streamlit and Rembg to remove the background from images (JPG, JPEG, PNG) and generate transparent images.

------------------------------------------------------
📦 Dependencies:
------------------------------------------------------
- streamlit
- rembg
- pillow (PIL)

Install using:
pip install streamlit rembg pillow

------------------------------------------------------
🚀 How to Run:
------------------------------------------------------
1. Make sure Python is installed.
2. Open terminal or command prompt.
3. Navigate to the project directory.
4. Run the app using:

   streamlit run app.py

(app.py should be the name of your Python file)

------------------------------------------------------
📁 Features:
------------------------------------------------------
✅ Upload JPG, JPEG, PNG images  
✅ AI-powered background removal  
✅ See original and result images  
✅ Download transparent PNG output  
✅ Clean, responsive UI  
✅ Footer with author credits  
✅ Fixed layout width: 1400px
